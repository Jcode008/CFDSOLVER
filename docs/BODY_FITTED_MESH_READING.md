# Body-Fitted Mesh Generation - Essential Reading List

## Foundation Papers (Read These First)

### 1. **Thompson, J.F., Warsi, Z.U.A., and Mastin, C.W. (1985)**
   *"Numerical Grid Generation: Foundations and Applications"*
   - **The Bible** of grid generation
   - Chapter 5: Elliptic generation equations (the math we're using)
   - Chapter 7: C-grids for airfoils specifically
   - **Download**: Free PDF available at: https://www.ma.ic.ac.uk/~jnc/CFD/References/thompson_1985.pdf
   - **Read**: Pages 145-189 (Elliptic methods), Pages 234-267 (Airfoil grids)

### 2. **Sorenson, R.L. (1980)**
   *"A Computer Program to Generate Two-Dimensional Grids About Airfoils and Other Shapes by the Use of Poisson's Equation"*
   - NASA Technical Memorandum 81198
   - **The practical implementation** - shows actual code structure
   - Covers the inverse Poisson formulation we need
   - Control functions P, Q for orthogonality
   - **Download**: https://ntrs.nasa.gov/citations/19800015694
   - **Read**: Full paper (35 pages) - very readable!

### 3. **Steger, J.L. and Sorenson, R.L. (1979)**
   *"Automatic Mesh-Point Clustering Near a Boundary in Grid Generation with Elliptic Partial Differential Equations"*
   - Journal of Computational Physics, Vol. 33, pp. 405-410
   - **How to cluster cells** near airfoil surface for boundary layer
   - Short and essential
   - **Download**: https://doi.org/10.1016/0021-9991(79)90034-0

## Advanced Topics (After Initial Implementation)

### 4. **Thomas, P.D. and Middlecoff, J.F. (1980)**
   *"Direct Control of the Grid Point Distribution in Meshes Generated by Elliptic Equations"*
   - AIAA Journal, Vol. 18, No. 6, pp. 652-656
   - Better control functions for orthogonality at walls
   - **Key for accuracy** in boundary layer regions
   - **Download**: https://doi.org/10.2514/3.50801

### 5. **Hilgenstock, A. (1988)**
   *"A Fast Method for the Elliptic Generation of Three-Dimensional Grids with Full Boundary Control"*
   - Numerical Grid Generation in CFD, Pineridge Press
   - Multi-grid acceleration (makes generation 10× faster)
   - **Read after** basic implementation works

## Airfoil-Specific Grid Topology

### 6. **Chan, W.M. and Steger, J.L. (1992)**
   *"Enhancements of a Three-Dimensional Hyperbolic Grid Generation Scheme"*
   - Applied Mathematics and Computation, Vol. 51, pp. 181-205
   - C-grid vs O-grid vs H-grid comparisons
   - **Why C-grid is best** for airfoils (especially at high α)

### 7. **Anderson, J.D. (2017)**
   *"Fundamentals of Aerodynamics" (6th Edition)*
   - Chapter 3: Conformal mapping for airfoils
   - Not grid generation, but **essential context** for understanding why orthogonality matters
   - Your library probably has this

## Practical Implementation Guides

### 8. **Spekreijse, S.P. (1995)**
   *"Elliptic Grid Generation Based on Laplace Equations and Algebraic Transformations"*
   - Journal of Computational Physics, Vol. 118, pp. 38-61
   - **Hybrid approach**: Start algebraic, then smooth with elliptic
   - Exactly what we'll implement!
   - **Download**: https://doi.org/10.1006/jcph.1995.1079

### 9. **Soni, B.K. (1992)**
   *"Grid Generation for Internal Flow Configurations"*
   - Handbook of Grid Generation, CRC Press, Chapter 18
   - Deals with **wake cut** treatment (our ξ-boundary issue)
   - Periodic vs matched wake conditions

## Modern CFD Textbooks with Grid Chapters

### 10. **Ferziger, J.H. and Perić, M. (2020)**
   *"Computational Methods for Fluid Dynamics" (4th Edition)*
   - Chapter 8: Grid generation
   - **Modern perspective** on structured vs unstructured
   - Code snippets in pseudo-code

### 11. **Tannehill, J.C., Anderson, D.A., and Pletcher, R.H. (1997)**
   *"Computational Fluid Mechanics and Heat Transfer" (2nd Edition)*
   - Chapter 5: Coordinate transformations
   - **All the metric derivative formulas** in one place
   - Good for reference while coding

## Key Equations Summary (From Papers)

### Inverse Poisson (Sorenson 1980, Eq. 15-16):
```
α·x_ξξ - 2β·x_ξη + γ·x_ηη + J²(P·x_ξ + Q·x_η) = 0
α·y_ξξ - 2β·y_ξη + γ·y_ηη + J²(P·y_ξ + Q·y_η) = 0

where:
α = x_η² + y_η²
β = x_ξ·x_η + y_ξ·y_η
γ = x_ξ² + y_ξ²
J = x_ξ·y_η - x_η·y_ξ
```

### Orthogonality Control (Thomas & Middlecoff 1980, Eq. 7-8):
```
P = -∂/∂ξ[J²·g^ηη·√(g_ξξ)]
Q = -∂/∂η[J²·g^ξξ·√(g_ηη)]

where g_ij are metric tensor components
```

### Clustering Function (Steger & Sorenson 1979, Eq. 3):
```
P = -a·exp(-b·η)·sinh(c·η)
```
(a, b, c control clustering strength)
